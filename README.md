# ReadScape
ReadScape is a modern web application that allows users to explore, search, and discover books using real-time data from a public API.

The application provides an intuitive and visually appealing interface where users can find books based on their interests, filter results, and view detailed information — all in a seamless experience.

---

# Features
* Smart Search
  Search books by title, author, or keywords in real-time.

* Advanced Filtering
  Filter books based on categories, authors, or publication year.

* Sorting Options
  Sort books alphabetically or by published date.

* Detailed Book View
  View complete information about each book:
  * Title
  * Author(s)
  * Published Date
  * Description
  * Cover Image

* Responsive Design
  Optimized for mobile, tablet, and desktop screens.
